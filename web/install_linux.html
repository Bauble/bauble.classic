Installing Bauble on Linux

(this page is a work in progress)

Install gtk, pygtk using the packaging tool(apt, rpm or emerge, etc..) for your distribution whether it be.

Choose a database engine. Download and install the database engine and it's python connector. These can most like be installed using the packaging tools for your distribution of choice. You can choose from the following database engines:

Database Connector
SQLite SQLite connector
MySQL mysql-python connector
Postgres psycopg connector
Firebird/Interbase KInterbaseDB connector
SapDB/MaxDB sapdb connector
Sybase Sybase connector

Download and install SQLObject

Download and install Bauble.
