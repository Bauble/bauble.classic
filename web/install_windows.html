Installing Bauble on Windows

Disclaimer: This doesn't describe how to setup and install a database server. If you don't know how to do that then you can use SQLite as the database backend to Bauble.

SQLite is a file based database and though it won't have the same features as the other databases such as networking and multiuser support it may be all you need to manager your collections.

  1. Download and install the latest GTK+ runtime from gladewin32.sourceforge.net. It should have a link that says Gtk+/Win32 Runtime Environment Installer <version> where <version> is the latest version available.
  2. Download the bauble-<version>-setup.exe installer where <version> is the latest version available. Hint: It will be the only installer available
  3. Double-click on the file you just downloaded to install Bauble
  4. Run Bauble from the Windows Start Menu under Bauble

Notes

Bauble comes with SQLite included. SQLite is a lightweight, file-based relational database. The simplest way to get started using Bauble is to use the SQLite backend.

To use other types of database backends you must install the database engine and a python connector for the database. The following is a list of database engines that Bauble supports and their connectors:

Database Connector
MySQL mysql-python connector
Postgres psycopg connector
Firebird/Interbase KInterbaseDB connector
SapDB/MaxDB sapdb connector
Sybase Sybase connector
